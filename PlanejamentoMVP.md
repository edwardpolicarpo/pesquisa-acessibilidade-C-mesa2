# 📝 Documento de Escopo: Loja de Action Figures

Este documento detalha as responsabilidades e a divisão de tarefas da equipe para as próximas etapas de desenvolvimento do e-commerce de Action Figures.

## 1. Visão Geral do Projeto
O objetivo é desenvolver uma plataforma robusta e acessível para a venda de colecionáveis, focando em performance (JS Vanilla) e experiência do usuário (UX/A11y).

---

## 3. Detalhamento das Etapas

### 🎨 Prototipação UI/UX (Figma)
* **Responsável:** Felipe Battalhini Boregio
* **Escopo:**
    * Criação do Wireframe e Protótipo de Alta Fidelidade.
    * Definição da paleta de cores, tipografia e componentes (Design System).
    * Mapeamento da jornada de compra (Home -> Busca -> Carrinho -> Checkout).

### 💻 Desenvolvimento Frontend (JS Vanilla + A11y)
* **Responsável:** Eduardo Carbonaro Policarpo
* **Escopo:**
    * Codificação da interface sem o uso de frameworks pesados (foco em performance).
    * Implementação de semântica HTML5.
    * Garantia de acessibilidade (suporte a leitores de tela, navegação por teclado e contraste).

### ⚙️ Backend e Banco de Dados
* **Responsáveis:** Davi Arakaki Oliveira de Souza e Lorraine Pinheiro Frontelli
* **Escopo:**
    * Modelagem da estrutura de dados (Produtos, Usuários, Pedidos).
    * Desenvolvimento de rotas CRUD (Create, Read, Update, Delete).
    * Integração entre o banco de dados e a interface frontend.

### 📄 Documentação Contínua
* **Responsável:** Davi Arakaki e Eduardo Carbonaro Policarpo
* **Escopo:**
    * Atualização constante do repositório (README.md).
    * Documentação das decisões arquiteturais.
    * Criação de manuais de uso e especificações da API.
